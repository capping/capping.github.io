## 1. 基本数据类型
- 数值型
  - 整数类型(byte, short, int, long)
  - 浮点类型(float, double)
- 字符型(char)
- 布尔型(boolean)

```
byte b = 2;
short s = 30000;
int i = 10;
long l = 4l;

char ch = '4';

boolean bool = true;
```
