1. 显示方式设置成表格
html:
```
<div id="wrapper">
    <div id="cell">
        <div class="content">Content goes here.</div>
    </div>
</div>
```
css:
```
#wrapper {
    display: table;
    height: 100px;
    background: pink;
}
#cell {
    display: table-cell;
    vertical-align: middle;
}
```
